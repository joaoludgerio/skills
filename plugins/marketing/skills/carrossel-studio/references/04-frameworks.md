# 04 — Frameworks e estruturas

## Tipos de carrossel
| Tipo | Quando | Estrutura típica |
|---|---|---|
| **Educativo** | ensinar um conceito/processo | gancho → contexto → passos → recap → CTA |
| **Lista** | "N coisas / N erros / N dicas" | gancho → 1 item por slide → fechamento |
| **Case** | provar com resultado | situação → problema → o que fez → resultado → lição |
| **Contraintuitivo** | quebrar uma crença | crença comum → "na real é o contrário" → prova → novo caminho |
| **Storytelling** | engajar pela narrativa | cena → tensão → virada → aprendizado → CTA |
| **Anúncio** | lançar algo | dor → solução → o que é → prova → CTA forte |

## Anatomia (5 a 10 slides)
1. **Slide 1 — Gancho** (`tipo: capa`). Só o título que para o dedo + "Arraste →". Sem corpo, ou 1 linha.
2. **Slides do meio — Desenvolvimento** (`tipo: conteudo`). Um ponto por slide. `kicker` numera ("Erro 1", "Passo 2") quando faz sentido.
3. **Último slide — Fechamento** (`tipo: cta`). Recap curto + chamada pra ação (salvar, seguir, comentar, link na bio).

## Regras de ritmo
- **5-7 slides** é o ponto doce. Acima de 10, a pessoa desiste.
- Alterne o peso: gancho forte → slides de respiro → ponto mais denso → fechamento leve.
- O penúltimo slide pode criar **antecipação** do fechamento ("e o erro mais comum é esse →").
- Se virou listão chato, corte. Menos slides bem feitos > muitos rasos.

## Quantos slides por tipo (sugestão)
- Lista de 3 → 5 slides (capa + 3 + cta)
- Educativo → 6-8 slides
- Case → 6 slides
- Contraintuitivo → 5-6 slides

Sempre proponha a estrutura e **valide antes de escrever a copy final**.
